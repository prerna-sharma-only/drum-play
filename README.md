## ✨ Overview

An interactive **drum kit web application** that allows users to play different drum sounds using **keyboard keys or mouse clicks**.

This project is designed to demonstrate **event handling, DOM manipulation, and audio integration** in a fun and engaging way.

---

## 🚀 Features

* 🎵 Play drum sounds using keyboard keys
* 🖱️ Click buttons to trigger sounds
* ⚡ Fast and responsive interface
* 🎨 Clean and minimal UI
* 💻 Optimized for desktop interaction

---

## 🛠️ Tech Stack

| Technology | Purpose                |
| ---------- | ---------------------- |
| HTML       | Structure              |
| CSS        | Styling & layout       |
| JavaScript | Logic & sound handling |

---

## 🎮 How to Use

```bash
1. Open the website
2. Press keys like (w, a, s, d)
   OR
3. Click on drum buttons
4. Enjoy the sound 🎶
```
