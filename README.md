🥁 Drum Play Website

An interactive drum kit web application that lets users play different drum sounds using keyboard keys or mouse clicks. Built for fun, learning, and practicing basic web development concepts.

🚀 Features
🎵 Play drum sounds with keyboard keys
🖱️ Click buttons to trigger sounds
⚡ Fast and responsive UI
🎨 Simple and clean design
📱 Works on desktop (keyboard support)
🛠️ Tech Stack
HTML – Structure of the webpage
CSS – Styling and layout
JavaScript – Logic and sound interaction
🎮 How to Use
Open the website
Press the keys shown on screen (like w, a, s, etc.)
OR
Click on the drum buttons
Enjoy the drum sounds 🎶
